# CPP_Projects

Collection of C++ projects focused on visual simulations and performance-friendly physics.

## Stack (Linux)

- Language: C++17
- Build: CMake
- Rendering: SFML

## Projects

- `gravity_nbody`: 2D N-body gravity simulator with presets, trails, and parameter controls.
- `atoms_md`: 2D molecular dynamics (Lennard-Jones) with temperature and density controls.

## Notes

Each folder is independent and can be compiled/run on its own.
